Управление курсором производится с помощью стрелок.
Для увеличения, уменьшения максимальной скорости соперников используются клавиши *, / соответственно. 
Для увеличения, уменьшения числа соперников используются клавиши +, - соответственно. 
Для перезапуска игры используется клавиша “end”. Для победы необходимо захватить 70% поля.
